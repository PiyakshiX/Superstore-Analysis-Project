# 📊 Superstore Sales Analytics & Prediction

An end-to-end Data Analytics and Machine Learning project built using the Superstore dataset. This project performs data analysis, visualization, dashboard creation, and sales prediction through a Streamlit web application.

## 🚀 Project Overview

This project demonstrates a complete data analytics workflow:

- 📊 Exploratory Data Analysis (EDA)
- 📈 Data Visualization
- 📉 Power BI Dashboard
- 🤖 Machine Learning for Sales Prediction
- 🌐 Streamlit Web Application

The application predicts sales based on user inputs such as quantity, discount, category, region, ship mode, and more.
