# Superstore Business Analytics Project

## Project Overview
This project analyzes the Superstore dataset using Python & Power BI.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Power BI

## Project Components

### Python EDA
- Data Cleaning
- Sales Analysis
- Profit Analysis
- Customer Analysis
- Discount Analysis

### Power BI Dashboard
- KPI Cards
- Sales Dashboard
- Profit Dashboard
- Customer Dashboard
- Interactive Slicers

## Key Insights
- West region generated highest sales and profit.
- Technology category was the most profitable.
- Consumer segment contributed maximum sales.
- High discounts negatively affected profit.
